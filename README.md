# uiuc-lifestyle

## Features
- User Login
- Bus Tracking + Class Schedule
- Grocery Best Prices
- Study Spaces
- Haircut Locations
- And more...

## Getting Started
1. Set Up Virtual Environment

   ```bash
   python3 -m venv virtualenv
   source virtualenv/bin/activate
   ```
   
2. Run Expo Go Frontend

   ```bash
   source virtualenv/bin/activate
   cd frontend
   npx expo start
   ```
